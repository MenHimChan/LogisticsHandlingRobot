## 2. Environment 🕹

The project is based on the following environment:

+ STM32 F103 ZET6 (Standard Peripheral Library)
+ OPENMV4 H7
+ 42 StepMotor + FOC
+ Metal Servo + Hiwonder - Servo Control board 
+ ......

All of out hardware information can be found in `doc` directory.
